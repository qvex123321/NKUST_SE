# 說明

<p>第一次開啟APP，必須先設定當前帳戶資料

<p>按下"+"號新增<img src="https://github.com/rabbit860321/NKUST_SE/blob/master/%E8%AA%AA%E6%98%8E%E5%9C%96%E7%89%87/%E6%96%B0%E5%A2%9E%E5%B8%B3%E6%88%B6.jpg" width="30%" height="30%"></p>
<p>新增完成後按下完成，進入主頁面<img src="https://github.com/rabbit860321/NKUST_SE/blob/master/%E8%AA%AA%E6%98%8E%E5%9C%96%E7%89%87/%E5%B8%B3%E6%88%B6%E6%96%B0%E5%A2%9E%E5%AE%8C%E6%88%90.jpg" width="30%" height="30%"></p>
<p>主頁面下方可以看到目前的帳戶金額，按下"+支出"可以新增消費紀錄<img src="https://github.com/rabbit860321/NKUST_SE/blob/master/%E8%AA%AA%E6%98%8E%E5%9C%96%E7%89%87/%E4%B8%BB%E8%A6%81%E4%BB%8B%E9%9D%A2.jpg" width="30%" height="30%"></p>
<p>輸入消費金額，按下"OK"，進入選擇項目頁面<img src="https://github.com/rabbit860321/NKUST_SE/blob/master/%E8%AA%AA%E6%98%8E%E5%9C%96%E7%89%87/%E8%BC%B8%E5%85%A5%E9%87%91%E9%A1%8D.jpg" width="30%" height="30%"></p>
<p>在選擇項目頁面中，點選左邊的主要項目，右邊會跳出該主要項目的副項目<img src="https://github.com/rabbit860321/NKUST_SE/blob/master/%E8%AA%AA%E6%98%8E%E5%9C%96%E7%89%87/%E9%BB%9E%E9%81%B8%E9%A0%85%E7%9B%AE.jpg" width="30%" height="30%"></p>
<p>也可以自己新增副項目，點選"+編輯"<img src="https://github.com/rabbit860321/NKUST_SE/blob/master/%E8%AA%AA%E6%98%8E%E5%9C%96%E7%89%87/%E6%96%B0%E5%A2%9E%E5%89%AF%E9%A0%85%E7%9B%AE.jpg" width="30%" height="30%"></p>
<p>按下確定，新增完成<img src="https://github.com/rabbit860321/NKUST_SE/blob/master/%E8%AA%AA%E6%98%8E%E5%9C%96%E7%89%87/%E5%AE%8C%E6%88%90%E6%96%B0%E5%A2%9E%E9%A3%B2%E6%96%99.jpg" width="30%" height="30%">
